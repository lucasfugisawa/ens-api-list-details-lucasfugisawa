Consultar uma api que retorna uma lista e depois buscar mais detalhes para cada resultado encontrado.

Sua função deverá consultar 2 APIs (que são promises) com dados de monstros do D&D 5e, representadas pelas funções abaixo:

  • listMonsters(query: string) - Retorna uma lista de monstros que contenham no nome o texto da query. Cada monstro retornado na lista inclui o campo index;
  • getMonster(index: string) - Pega os detalhes de um monstro de acordo com o seu index;

Sua função deve receber um nome e buscar a lista de todos os monstros com esse nome. Para cada monstro encontrado, pegar os dados completos do monstro. Sua função deve devolver um array em que cada posição possui um objeto com o nome do monstro (name) e o valor de vida do monstro (campo hit_points)

Exemplo de resposta:

  await getMonstersHitPoints("ogre");
  [
    {name: "Ogre", hit_points: 59},
    {name: "Ogre Zombie", hit_points: 85},
  ]

Dicas:

  • Promises são encadeáveis;
  • Promise.all;
